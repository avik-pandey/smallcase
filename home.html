Welcome to the Smallcase Portfolio Tracking API

List of APIs

Add a security : /api/security/add

Update a security : /api/security/update/:tickerSymbol

Delete a security : /api/security/remove/:tickerSymbol

Fetch the list of all securities : /api/security/fetch

Add a new trade : /api/trade/add

Update a trade : /api/trade/update/:tradeId

Delete a trade : /api/trade/remove/:tradeId

Fetch the list of all trades grouped by tickerSymbol : /api/trade/fetch

Fetch the portfolio : /api/portfolio/fetch

Get the returns at the time of calling the API : /api/portfolio/returns
